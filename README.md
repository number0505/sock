#Steps to Run the Website


1. Download the complete code from GitHub.

2. Open the downloaded code in Visual Studio Code.

3. In the terminal at the bottom of Visual Studio Code, enter the following commands in order,

	a. install node

	b. npm install

	c. export GOOGLE_APPLICATION_CREDENTIALS="./auth/veraAuthKey.json"

	d. node index.js

After completing these steps, the website should be up and running.
website: http://localhost:8081/
